# Mi300X Bandwidth Test

Memory bandwidth test suite for MI300X

## Features

- Optimized for AMD Instinct MI300X accelerators
- ROCm 6.0+ compatible
- Python 3.8+ with PyTorch ROCm support
- JSON output for CI integration

## Requirements

- AMD GPU with ROCm 6.0+
- Python 3.8+
- PyTorch with ROCm support

## Quick Start

```bash
git clone https://github.com/iati146/mi300x-bandwidth-test.git
cd mi300x-bandwidth-test
pip install -r requirements.txt
python main.py --benchmark
```
